Word Float

About

このサイトは、最新ニュースを取得し、単語の出現頻度を可視化するWebアプリケーションです。単語は出現頻度が高いほど大きく表示され、物理演算によって画面内を漂います。

How to Use

基本操作

  • オブジェクトの操作

    Canvas内に表示されている各単語（オブジェクト）は、マウスクリックでドラッグすることにより、任意の位置へ移動させることが可能です。

  • 関連記事の参照

    特定の単語オブジェクトをクリックすると、その単語を含むニュース記事が画面下部の "Articles" セクションでハイライト表示され、リストの先頭に並び替わります。記事カードをクリックすることで、元のニュースソースを新しいタブで開きます。

  • キー操作

    Canvasが表示されている状態で以下のキーを押すと、全単語をまとめて操作できます（1回押すごとに1回適用。強く動かしたいときは連打してください）。

    • W / A / S / D：全単語を 上 / 左 / 下 / 右 方向へ押し出します。

    • F：全単語を加速します。動いている単語は現在の進行方向へ、静止している単語はランダムな方向へ加速します。

    • R：全単語をランダムな位置に再配置し、速度を初期状態に戻します（ニュースの再取得は行いません）。

設定パネル

画面左上のメニューボタンから設定パネルへアクセスできます。各パラメータはスライダーのドラッグ、または数値ボックスへの直接入力で設定できます。変更後は "Reload" ボタンを押すことでシミュレーションに反映されます。

  • Edition

    取得するニュースの国・言語エディションを指定します（日本 / 米国）。

  • Category

    取得するニュース記事のカテゴリ（トピック）を指定します。

  • Top N

    出現頻度が高い上位N個の単語を抽出して表示します。数値を大きくすると、より多くの単語が表示されます。

  • Font Size (文字サイズ)

    単語の文字サイズ（ピクセル）の最小値・最大値を範囲スライダーで指定します。出現頻度が最も低い単語が最小値、最も高い単語が最大値のサイズで表示され、その間は頻度に応じて補間されます。

  • Air Resistance (空気抵抗)

    オブジェクトの動きに対する空気抵抗の強さを設定します。値が小さいほど減速しにくく漂い続け、大きいほど早く止まります。

  • Restitution Coefficient (反発係数)

    オブジェクトが他のオブジェクトや壁と衝突した際の反発の強さを設定します。値が1のとき完全弾性衝突に近くなり、高く跳ね返ります。

  • Max Speed (最大速度)

    オブジェクトが移動できる速度の上限を設定します。値を大きくすると、より速く飛び回るようになります。

  • Rotation (回転)

    単語オブジェクトが物理演算で回転するかどうかを切り替えます。オフにすると文字は常に水平のまま漂います。変更後、"Reload" ボタンを押すことで反映されます。

  • Excluded Word (除外単語)

    分析対象から除外したい単語を指定します。入力フィールドに単語を入力しEnterキーを押すことで、リストに追加されます。一般的な助詞・接続詞・ストップワードはバックエンドで自動的に除外されます。
